# Pasha Platform — Release Notes

**Version:** 1.0
**Release Date:** July 1, 2026

---

## About This Release

This is the first full release of the Pasha platform — a complete startup ecosystem covering a public-facing website, an applicant application portal, a committee review workspace, and a powerful admin back office. This document summarizes everything that is now available.

---

## Public Website

- Modern, fully responsive website with Home, About, Directory, and Events pages
- Public startup directory with individual startup profile pages
- Startup profiles display logos, cover images, and earned badges
- Dedicated women-led startups showcase
- Events listing with individual event detail pages
- Built-in live chat widget for visitor questions
- Polished, premium design across the entire site

## Applicant Portal

- Simple account sign-up with email verification
- Smart prompt to sign in when an email is already registered
- Forgot password and password reset support
- Dynamic application form supporting text, rich text, dropdowns, and file uploads
- Save progress as a draft and return to finish later
- Clear application status tracking through a 6-stage workflow
- Helpful confirmation messages after key actions
- Fully optimized for mobile devices

## Committee Workspace

- Invite committee members and guide them through onboarding
- Dedicated committee portal for reviewing applications and startups
- Committee activity tracking with clear, readable history
- Streamlined member verification
- Quick access to the committee area from the home page

## Admin Dashboard

- Flexible form builder to create and edit application forms without developer help
- Central submissions view with search and pagination
- Searchable databank of all applicant and startup records
- One-click data export to CSV for reporting
- Curate and manage featured startups
- Manage reusable dropdown/option lists shared across forms
- Full events management
- Support for importing and displaying legacy/historical data
- Protected system fields to prevent accidental edits

## Email System

- Branded email template editor
- Consistent, professional header and footer across all emails
- Broadcast tool to send announcements to applicants or committee members
- Email activity log for tracking sent messages
- Automated emails for verification, password resets, and confirmations

## Administration & Security

- Secure admin login, logout, and password reset
- Separate super-admin area for top-level platform control
- Reliable session handling, including automatic logout when sessions expire



## Looking Ahead

This release establishes the complete foundation of the Pasha platform. Future updates will build on it based on your feedback and evolving priorities.
